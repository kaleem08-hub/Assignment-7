Instructions for using the online calculator

Keys function

[ 0 ], [ 1 ], [ 2 ], ... [ 9 ] - standard number keys.
[ 00 ] - key input 2 zeros.
[ + ]addition, [ - ]subtraction, [ х ]multiplication, [ ÷ ]division.
[ % ] - calculates percentages.
[ √ ] - calculate the square root.
[ C ] - clear all input to the calculator.
     - a way to erase or clear an entry.
[ log ] - It allows you to calculate natural logarithm of any number.
[ π ] - displays the value of pi (the ratio of the circumference of a circle to its diameter).
[ xy ] - calculation of X to the power of Y.
[ ex ] - returns the value of e x , where e is euler's number (approximately 2.7183) and x is the number passed to it.
